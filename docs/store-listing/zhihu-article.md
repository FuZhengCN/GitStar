# 做了个浏览器插件，专门用来逛 GitHub

GitHub 的搜索是给知道自己要找什么的人用的，但我只想逛。

点开看看最近有什么好项目，看看这周谁在火，看看 Python 又出了什么新玩意——不需要敲 `stars:>1000 language:python created:>2026-05-28` 这种咒语。

**GitStar，弹窗里就能逛。**

---

**能干什么：**

- 🔥 **热门 / 🚀 崛起 / 📈 活跃** 三种模式，一键切换不同的逛法。崛起模式按日均 Star 增长率排序，专门抓爆款
- 语言、时间、排序，三个下拉框搞定，不需要学搜索语法
- 弹窗里直接看 README，预览 + 目录导航，不用跳标签页
- 一键 AI 概述：三张卡片看懂一个项目（功能 / 特点 / 场景）
- 本地收藏夹，独立于 GitHub Star，数据在你本地
