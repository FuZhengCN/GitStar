# GitStar · GitHub 热门项目发现 — Discover Trending GitHub Repositories

## 中文

GitStar 帮助你在浏览器工具栏中快速发现 GitHub 上的优质高星开源项目。

为什么安装 GitStar？

GitHub 自带的搜索和探索功能是为明确知道自己要找什么的开发者设计的。如果你想按编程语言、时间段或热门程度发现好项目，就需要手动编写复杂的搜索查询。GitStar 用一个简洁快速的界面帮你完成这一切。

功能一览：

• 三种发现模式 — 一键切换热门（总 Star 数）、新星（周增长率排名）、活跃（近期更新）。新星模式在每个项目上显示每日增长星数徽章。
• 搜索筛选 — 按关键词、编程语言、创建时间（本周 / 本月 / 今年）查找项目，支持按 Star、增长率、Fork、最近更新排序。几次点击即可，无需学习搜索语法。
• 即时查看 README — 在弹窗中直接查看渲染后的 README 文档，支持目录导航。大文件分段加载，界面保持流畅。
• 一键 Star — 从详情页一键 Star / Unstar 仓库（需配置 GitHub Personal Access Token）。
• 本地收藏 — 将项目保存到本地收藏夹，完全独立于 GitHub 的 Star 体系。收藏数据仅存储在你的设备上。
• 侧边栏推荐 — 浏览任意 GitHub 仓库页面时，GitStar 在侧边栏注入一个推荐面板，根据 topic 相似度展示同类热门项目。支持折叠、拖拽、适配深色模式。
• AI 智能概述 — 一键生成项目的三字段结构化摘要（功能·特点·场景），用一张卡片快速理解项目。支持自定义 API 端点、模型和输出语言。
• 双语界面 — 完整的中英文支持，语言偏好在弹窗、选项页、侧边栏之间实时同步。

适合谁用？

开发者、学生和技术爱好者，想持续关注开源生态但不想花时间编写搜索查询。无论你是在学习新语言、为下一个项目评估库，还是单纯好奇社区正在构建什么，GitStar 都能让发现变得毫不费力。

## English

GitStar helps you discover high-quality, high-star GitHub projects — right from your browser toolbar.

Why install GitStar?

GitHub's own search and explore features are designed for developers who already know what they're looking for. If you want to discover great open-source projects by language, time period, or popularity, you have to craft complex search queries. GitStar does this for you with a clean, fast interface.

What you can do:

• Three discovery modes — Switch between Hot (all-time stars), Rising (weekly growth rate ranking), and Active (recently updated). Rising mode shows a stars/day badge on each project.
• Search & filter — Find projects by keyword, programming language, creation time (this week / month / year), and sort by stars, growth rate, forks, or recent updates. All with a few clicks, no query syntax needed.
• Read README instantly — View rendered README directly in the popup with a table of contents overlay. Large files load progressively so the UI stays responsive.
• Star directly — Star or unstar repositories with one click from the detail page (requires GitHub personal access token).
• Bookmark locally — Save projects to your local favorites, completely independent of GitHub's star system. Your bookmarks stay private on your device.
• AI-powered overview — Generate a structured three-section summary (Features · Highlights · Use Cases) of any project's README with one click. Understand a project at a glance. Configurable API endpoint, model, and output language.
• Bilingual UI — Full Chinese and English support, with language preference synced across popup and options.

Who is this for?

Developers, students, and tech enthusiasts who want to stay on top of the open-source ecosystem without spending time crafting search queries. Whether you're learning a new language, evaluating libraries for your next project, or just curious about what the community is building, GitStar makes discovery effortless.
