# GitStar 图标重新设计 — Design Spec

**日期:** 2026-05-28  
**状态:** 已确认

## 设计目标

- G 字母代表 GitHub 平台
- 金色星标代表有价值的开源项目
- 整体和谐、专业、清爽
- 在 16/32/48/128px 各尺寸下均清晰可辨
- 纯色填充为主，避免 Plasmo SVG 渲染器的渐变/色偏缺陷

## 最终方案

**蓝底白G + 金星徽章**

```
┌──────────────────────┐
│  ████████████    ★   │
│  ████████████        │
│  ████  G  ████       │
│  ████████████        │
│  ████████████        │
│                      │
│                 G    │
└──────────────────────┘
```

| 元素 | 说明 | 色值 |
|------|------|------|
| 圆角方底 | 品牌蓝，rx=28 | `#3b82f6` |
| G 字母 | 白色粗体几何无衬线，font-size=88, weight=900 | `#ffffff` |
| 星标 | 右上角五角星，纯色填充 | `#f59e0b` |

G 字母文字基线位于 (54, 104)，视觉重心约在 (54, 73)，落在图标左下→右上对角线（y = 128 - x）上，与右上角星标形成对角平衡。

## 构图原理

- **对角线平衡**: 星标 (101, 33) 在右上区域，G 重心 (54, 73) 在左下区域，两者沿对角线对称
- **主次分明**: G 为主体（代表 GitHub），星标为点缀（代表优质项目），不是平分秋色
- **纯色安全**: 背景 #3b82f6、文字 #ffffff、星标 #f59e0b 均为纯色，不依赖渐变，Plasmo 渲染兼容

## 尺寸适配

| 尺寸 | G 辨识 | 星标辨识 | 整体效果 |
|------|--------|----------|----------|
| 16px | 白 G 在蓝底上可识别 | 星标细节丢失不影响整体辨识 | 蓝色圆角方块 + 白色图形 ✓ |
| 32px | 清晰可辨 | 星标可见 | 工具栏最佳尺寸 ✓ |
| 48px | 非常清晰 | 星标清晰 | 扩展管理页 ✓ |
| 128px | 完美 | 完美 | 商店展示 ✓ |
